## 🔐 Day 4 – Authentication & Frontend Integration

### 🔹 Topics Covered

* JWT (JSON Web Token)
* Authentication & Authorization
* Protected Routes
* Frontend–Backend Integration
* Token Storage (localStorage)

### 🛠️ Work Done

* Created Login API using JWT
* Generated token on successful login
* Implemented protected routes using Bearer Token
* Integrated frontend with backend APIs
* Stored token in localStorage after login
* Sent token in all API requests for secure access
* Restricted access to only authenticated users
* Verified complete flow:
  Frontend → Backend → Database → Response → UI
